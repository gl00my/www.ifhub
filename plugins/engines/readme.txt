ОПИСАНИЕ
Плагин добавляет новый тип топика "Движки (engines)" с полями для заполнения:
- заголовок
- выбор блога
- текст
- список тегов
- возможность принудительного вывода на главную (для админов)
- URL
- Платформы
Данный тип топика разрешено коментировать, управление доступно только админам, выводится в отдельный пункт меню, на страницы со списком топиков "Движки" добавлен стандартный сайдбар (прямой эфир, блоги, теги).

УСТАНОВКА
1. Скопировать плагин в каталог /plugins/
2. Через панель управления плагинами (/admin/plugins/) запустить его активацию.
3. Выполнить настройки плагина в файле /plugins/engines/config/config.php
4. Радоваться ;)
